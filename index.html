¿Qué es la Cadena de Custodia?

La cadena de custodia es el conjunto de procedimientos técnicos, documentales y legales que permiten garantizar la autenticidad, integridad y conservación de los indicios o evidencias desde el momento de su obtención hasta su presentación ante la autoridad judicial.

Cadena de Custodia en el Ecuador

En el Ecuador, la cadena de custodia es un elemento fundamental dentro del proceso penal y está regulada por normas y protocolos que buscan asegurar que la evidencia no sea alterada, contaminada o manipulada indebidamente. Su correcta aplicación garantiza el debido proceso y la validez probatoria de los indicios presentados ante jueces y tribunales.

Importancia de la Cadena de Custodia Digital

Con el avance de la tecnología y el incremento de delitos informáticos, la cadena de custodia digital se vuelve esencial para el manejo de evidencias electrónicas como archivos, registros, imágenes, videos o información almacenada en sistemas informáticos. Aclarar que todos los registros se guardaran de manera local, es decir en el mismo ordenador.

Beneficios de una Cadena de Custodia Digital

  • Garantiza la integridad y autenticidad de la evidencia digital.
  • Reduce errores humanos en el registro y control de evidencias.
  • Permite un mejor seguimiento y trazabilidad de los indicios.
  • Agiliza los procesos investigativos y judiciales.
  • Fortalece la validez legal de la prueba digital.

Guía de Instalación de la Aplicación

Primeros pasos

  1. Buscar en el navegador e instalar Visual Studio Code.
  2. Instalar Python (no instalar el Python Manager, solo la versión actual, por ejemplo Python 3.14).
  3. Descargar la carpeta que contiene los archivos de la aplicación.
CADENA DE CUSTODIA ARCHIVOS VISUAL STUDIO CODE

Reemplazo de la carpeta de Visual Studio Code

Una vez instalada la carpeta de Microsoft Visual Studio Code, se debe reemplazar la carpeta original:

  • Clic derecho sobre Visual Studio Code (la app).
  • Seleccionar Abrir ubicación del archivo.
  • La ruta normalmente será: C:\Users\user\AppData\Local\Programs
  • Copiar o arrastrar la carpeta Microsoft VS Code descargada.
  • Reemplazar la carpeta existente.

Esto se realiza para que las extensiones necesarias para ejecutar la aplicación ya se encuentren instaladas.

Instalación manual de extensiones (si es necesario)

  • HTML
  • CSS
  • Java
  • Python
  • (Opcional) Flask

Apertura de Visual Studio Code

Si Visual Studio Code no abre después del reemplazo, es normal. Ingrese a la carpeta: Programs > Microsoft VS Code y ejecute la aplicación llamada code.

Abrir el proyecto

  • En Visual Studio Code, ir a File.
  • Seleccionar Open Folder o Abrir carpeta.
  • Seleccionar la carpeta cadena_custodia_prototipo.

Configuración de Windows (Python)

Buscar el Administrador de alias de ejecución de aplicaciones:

  • Configuración > Aplicaciones > Aplicaciones y características.
  • Buscar Administrar alias de ejecución de aplicaciones.
  • Desactivar python.exe y python3.exe (AVISO IMPORTANTE).

Uso de la terminal en Visual Studio Code

En la parte inferior aparecerá la terminal. Si se ve pequeña, puede ajustarse arrastrando con el mouse.

Comandos a ejecutar en la terminal

python --version
python3 --version

python -m venv venv
venv\Scripts\activate

python -m pip install --upgrade pip
pip install flask
pip install flask-cors
pip install -r requirements.txt
pip list

cd backend

python app.py
flask run

El sistema mostrará un enlace similar a: http://127.0.0.1:5000

Este enlace abrirá la aplicación únicamente en el navegador local del ordenador.

Nota: Si todo está instalado correctamente y la aplicación no abre, se debe consultar con el creador de la aplicación.
